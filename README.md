# `jscli`
> A CLI for javascript

# Usage

```sh
npm install --global jscli
```

```
Usage: jscli <command>

Commands:
  init  Initialize a new JavaScript project
  ls    List scaffolds

Options:
  --version   Show version number                                      [boolean]
  -h, --help  Show help                                                [boolean]
```

## Scaffolds

To initialize a project run

```sh
js init <scaffold> <dirname>
```
